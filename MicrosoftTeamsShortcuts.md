# Microsoft Teams Shortcuts

| Shortcut              | Description                        | Alternative         |
| ---                   | ---                                | ---                 |
| `Ctrl`+`.`            | Show keyboard shortcuts            |                     |
| `Ctrl`+`Shift`+`O`    | Camera On/Off                      |                     |
| `Ctrl`+`Shift`+`M`    | Mute/Unmute                        |                     |
| `Ctrl`+`Shift`+`E`    | Share Content                      |                     |
| `Ctrl`+`Shift`+`V`    | Unformatted Paste                  |                     |
