# Miro Shortcuts

| Shortcut                  | Description                         |
| ---                       | ---                                 |
| `F1`                      | Miro shortcut list                  |
| `V` / `H`                 | Select tool / Hand mode             |
| `T`                       | Text                                |
| `N`                       | Open sticky notes on toolbar        |
| `S`                       | Open shapes on toolbar              |
| `R`                       | Rectangle                           |
| `O`                       | Oval                                |
| `L`                       | Connection line, arrow              |
| `D`                       | Card                                |
| `P`                       | Pen                                 |
| `E`                       | Eraser                              |
| `C`                       | Comment                             |
| `F`                       | Frame                               |
| `M`                       | Minimap                             |
| `Ctrl`+`K`                | Link to                             |
| `Ctrl`+`D`                | Duplicate                           |
