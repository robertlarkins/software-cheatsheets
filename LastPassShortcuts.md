# LastPass Shortcuts

| Shortcut            | Description                        | Alternative         |
| ---                 | ---                                | ---                 |
| `Alt`+`G`           | Generate secure password           |                     |
| `Alt`+`I`           | Recheck Page                       |                     |
| `Alt`+`W`           | Site Search                        |                     |
| `Alt`+`PgUp`        | Next Login (when multiple)         |                     |
| `Alt`+`PgDn`        | Previous Login (when multiple)     |                     |
| `Ctrl`+`Alt`+`H`    | Open LastPass Vault                |                     |
